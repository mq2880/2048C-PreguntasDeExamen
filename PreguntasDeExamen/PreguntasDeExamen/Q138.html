Pregunta 138

- display:inline;Incorrecta Muestra un elemento como un elemento en línea (como span). Cualquier propiedad de altura y ancho no tendrá efecto
display: value;

- box-sizing:border-box;Correcta La propiedad define cómo se calculan el ancho y el alto de un elemento. Con border-box las propiedades de ancho y alto (y propiedades mínimas / máximas) incluyen contenido, relleno y borde
box-sizing: content-box|border-box|initial|inherit;

- box-shadow:10px 10px 10x #f00; Incorrecta esta propiedad adjunta una o más sombras a un elemento y el ejemplo no lleva sombras
box-shadow: none|h-offset v-offset blur spread color |inset|initial|inherit;

- box-shadow:4px 4px 4x #f00; Incorrecta igual que la anterior


- border-radius:10px;Correcta La propiedad define el radio de las esquinas del elemento
border-radius: 1-4 length|% / 1-4 length|%|initial|inherit;hyphens: none|manual|auto|initial|inherit;

- border-spacing:10px;Incorrecta establece la distancia entre los bordes de las celdas adyacentes
border-spacing: length|initial|inherit;

- border-bottom-style:outlet;Incorrecta establece el estilo del borde inferior de un elemento.
border-bottom-style: none|hidden|dotted|dashed|solid|double|groove|ridge|inset|outset|initial|inherit;

- box-decoration-break:slice; Incorrecta especifica cómo se aplica el fondo, el relleno, el borde, la imagen del borde, la sombra del cuadro, el margen y la ruta de recorte de un elemento cuando el cuadro del elemento está fragmentado.
box-decoration-break: slice|clone|initial|inherit|unset;
